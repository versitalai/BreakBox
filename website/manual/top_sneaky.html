Version 2.0.0 | JUKEBOX LIVE⎋

"JukeBox Live" is the new hub for viewing and submitting new presets.

What are live submissions?

Presets submitted to JukeBox will now be added automatically every Sunday, independantly from JukeBox updates. This will be a sentence that explains the preset verification process.

Will this change how I submit presets?

Largely, no. You do have to make an account for the website, but after that, the process is nearly the same

Website design inspired by BeepBoard and Newgrounds
  • an external website that allows presets to be added to JukeBox as soon as they are verified
  • allows submitting multiple instruments in the same song
  • allows upvoting, downvoting, tag suggestions, name suggestions, and reporting
  • allows viewing pending presets (sorted by least engagement top, oldest top, so every instrument gets critiqued)
  • allows viewing a list of currents presets (with additional statistics and whatnot)

Note, there are some caviats to the new system:

  • The "JukeBox Preset Submission Form" is no longer available (the submission sheet is still up for archival purposes)
  • JukeBox now requires account creation to be able to submit presets
  • everything other than submitting instruments require account verification via discord
